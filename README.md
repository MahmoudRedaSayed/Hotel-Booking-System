# AlgoPhobia Booking System
it is hotel booking system you can browse the rooms and choose any room in any where to book it


## User's Features 
- Login and Register
- book room 
- pay the booking
- search in the room accordding to the booking date
## Admin's Features 
- View all (users , rooms ,   bookings)
- Add room

## Technologies
- express js
- Redux
- React 
- mongodb
- stripe (payment tool)
